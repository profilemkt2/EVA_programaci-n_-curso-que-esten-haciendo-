# EVA - Curso de Programación

## 📋 Descripción

Este repositorio corresponde al curso de programación del programa EVA. Aquí encontrarás todos los materiales, ejercicios y proyectos desarrollados durante el curso.

## 🎯 Objetivos del Curso

- Aprender los fundamentos de programación
- Desarrollar habilidades de resolución de problemas
- Practicar con ejercicios prácticos y proyectos reales
- Colaborar en equipo utilizando Git y GitHub

## 🚀 Cómo Usar Este Repositorio

### Prerrequisitos

- Git instalado en tu computadora
- Editor de código (VS Code, PyCharm, etc.)
- Lenguaje(s) de programación según el curso

### Clonar el Repositorio

```bash
git clone https://github.com/profilemkt2/EVA_programaci-n_-curso-que-esten-haciendo-.git
cd EVA_programaci-n_-curso-que-esten-haciendo-
```

### Realizar Cambios

1. Crea una rama para tus cambios:
   ```bash
   git checkout -b tu-rama-descriptiva
   ```

2. Realiza tus cambios y commits:
   ```bash
   git add .
   git commit -m "Descripción clara del cambio"
   ```

3. Envía tus cambios:
   ```bash
   git push origin tu-rama-descriptiva
   ```

4. Crea un Pull Request en GitHub
